# KeySwitchManager

KeySwitchManager は複数のDAW用キースイッチ定義ファイル作成に特化した、キースイッチ情報をテキストやスプレッドシートで編集、ローカルデータベースファイルで管理可能なツール及びライブラリアン。

使い方など、詳しい情報は wiki へ

- [Wiki](https://github.com/r-koubou/KeySwitchManager/wiki)

## 対応DAW

- Cubase (Pro), Nuendo
- Studio One 5.2 以降
- Cakewalk
